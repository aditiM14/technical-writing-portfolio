# TaskFlow Release Notes

## Version: 1.1

## Release Date: April 2026

## Overview

This release introduces new features and improvements to enhance user experience and performance. It also includes bug fixes to improve system reliability.

## New Features

- **Dark Mode**
Users can switch to dark mode for a more comfortable viewing experience.
- **File Attachments**
Users can now attach files (up to 10MB) to tasks.

## Improvements

- **Improved Search Performance**
Search results are now faster and more accurate.
- **Enhanced Mobile Responsiveness**
Improved layout and usability on smaller screens.

## Bug Fixes

- Fixed an issue where tasks were duplicated when saved multiple times.
- Fixed a delay in email notifications.

## Known Issues

- File uploads may fail on slow internet connections.
- Dark mode is not fully supported on older browsers.

## Upgrade Notes

No action is required. All updates are applied automatically.
