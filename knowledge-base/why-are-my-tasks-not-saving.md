# Why Are My Tasks Not Saving?

## Overview

This article helps you troubleshoot issues when tasks are not being saved in TaskFlow.

## Problem

You create or edit a task, but the changes are not saved.

## Solution

Try the following steps:

1. Ensure all required fields (such as task title) are filled.
2. Check your internet connection.
3. Refresh the page and try saving again.
4. Log out and log back in.
5. Try using a different browser.

If the issue continues, contact support.

## Additional Notes

- Task titles are mandatory and must not be empty.
- Slow internet may prevent changes from saving properly.
