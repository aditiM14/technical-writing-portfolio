# How to Create a Task

## Overview

This article helps you understand how to create a task in the TaskFlow project.

## Problem

You want to create a new task in a project.

## Solution

To create a task:

1. Click New Task.
2. Enter the task title.
3. Add a description and other details (optional).
4. Set a due date.
5. Click Save.

## Additional Notes

- Task titles are mandatory and must not be empty.
- Check your internet connection if the task details are not saving.
