# User Guide - TaskFlow

## 1. Introduction

**What TaskFlow is:**

TaskFlow is a web-based task management application that helps users organize, track, and manage their daily tasks and projects.

**Who it is for:**

It is designed for individuals, students, freelancers, and small teams who want a simple way to manage their work.

**What users can do:**

With TaskFlow, users can create tasks, set deadlines, organize projects, and collaborate with team members.

## 2. Getting Started

### 2.1 System Requirements

TaskFlow is a web-based application and does not require installation.

To use TaskFlow, ensure the following:

- A modern web browser (Google Chrome, Firefox, or Microsoft Edge)
- An active internet connection

### 2.2 Creating an Account

To create a new TaskFlow account:

1. Open the TaskFlow website.
2. Click Sign Up.
3. Enter your full name.
4. Enter your email address.
5. Create a password (minimum 8 characters).
6. Click Create Account.

After registration, a verification email will be sent to your email address.

### 2.3 Logging In

To log in to your account:

1. Open the TaskFlow website.
2. Click Log In.
3. Enter your registered email address.
4. Enter your password.
5. Click Log In.

If your credentials are correct, you will be redirected to the dashboard.

### 2.4 Dashboard Overview

After logging in, the dashboard displays:

- Sidebar navigation for accessing projects and tasks
- Task summary showing pending and completed tasks
- Upcoming deadlines
- Notifications panel

## 3. Managing Tasks

### 3.1 Creating a Task

To create a task:

1. Click New Task.
2. Enter the task title.
3. Add a description (optional).
4. Set a due date.
5. Click Save.

### 3.2 Editing a Task

To edit a task:

1. Open the task.
2. Click Edit.
3. Update the required fields.
4. Click Save.

### 3.3 Deleting a Task

To delete a task:

1. Open the task.
2. Click Delete.
3. Confirm the action.

### 3.4 Marking a Task as Complete

To mark a task as complete:

1. Open the task.
2. Click Mark as Complete.

### 3.5 Task Status

Tasks can have the following statuses:

- To Do
- In Progress
- Completed
- Archived

## 4. Managing Projects

### 4.1 Creating a Project

To create a project:

1. Click New Project.
2. Enter the project name.
3. Click Create.

### 4.2 Adding Tasks to a Project

To add a task to a project:

1. Open the project.
2. Click Add Task.
3. Enter task details.
4. Click Save.

### 4.3 Deleting a Project

To delete a project:

1. Open the project.
2. Click Delete Project.
3. Confirm the action.

## **5. Collaboration Features**

### 5.1 Inviting Team Members

To invite a team member:

1. Open a project.
2. Click Invite.
3. Enter the email address.
4. Click Send Invitation.

### 5.2 Assigning Tasks

To assign a task:

1. Open the task.
2. Select Assign User.
3. Choose a team member.

### 5.2 Assigning Tasks

Users can add comments to tasks to communicate with team members.

## 6. Notifications

Users receive notifications when:

- A task is assigned
- A task is marked complete
- A deadline is within 24 hours
- They are mentioned in a comment

Notifications appear in the app and via email (if enabled).

## 7. Search Function

To search for tasks:

1. Use the search bar at the top.
2. Enter a keyword.
3. Select the relevant result.

Users can search by:

- Task name
- Tag
- Project name
- Assigned user

## 9. Troubleshooting

**Issue: Unable to log in**

Solution: Check your email and password. Use “Forgot Password” if needed.

**Issue: Task not saving**

Solution: Ensure all required fields are filled and check your internet connection.

**Issue: Verification email not received**

Solution: Check spam/junk folder. Ensure you entered the correct email address.

**Issue: Unable to invite a team member**

Solution: Make sure the email address is correct. Accept the invitation within the time limit.
